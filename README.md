# bobolu-backend

## Commands

```bash
# install dependencies
$ npm install

# serve API at localhost:3030
$ node .

# serve API at 192.168.0.100:8080 (bash)
$ HOST=192.168.0.100 PORT=8080 node .
```

## Config

Use the following environment variables:

 - `PORT` - listening port
 - `HOST` - listening interface
